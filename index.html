Tic-Tac-Toe

I created my own original Tic Tac Toe game from scratch using HTML5, CSS, and JavaScript. There is a two player mode where you can play with a friend locally. There is also a single player mode where you play against the AI that I created just for the game. Don't get frustrated while playing it though... It's very difficult to beat.

Weather Map

Using HTML5, CSS, and JavaScript I created the outline for this weather map app. It incorporates both the Open Weather API as well as the Google Maps API. They are connected together to allow a user to drag a pin around on the map to be able to choose where they want to see the weather. The panels are then instantly updated with the new weather.
